Circumference and area calculator

Extra php program for assignment-2



Please note that the php aspect of the program will not run from the published github page.
Consult the two links below to run the program with the php aspect on mac.

Creating the files

Create the files on your mac using the instructions on the web page and copy and paste the code inside index.html, calculations.php and stylesheet.css inside three different files named extra.html, calculations.php and stylesheet.css. Put these two files into a folder.

Launching the php website

This explains how to launch the php website. Copy and paste "http://localhost:9000/extra.html" into a tab.

If there is an easier way to do this please let me know!! I spent 2 hours researching a way to run a php server :,)
